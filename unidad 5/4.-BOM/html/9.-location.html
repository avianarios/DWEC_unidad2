location

is reachable through window and document. Interesting properties:

  • hostname
  • href
  • protocol

Opening URLs

alert, confirm and prompt

  1. alert: Shows a message in a modal window
  2. confirm: Shows a confirmation window
  3. prompt: Asks for data

location properties

  1. href: entire URL
  2. protocol: protocol scheme of the URL, including the final ':'.
  3. host: hostname:port or the URL
  4. hostname: domain of the URL
  5. port: port number of the URL
  6. search: "querystring" of the URL
