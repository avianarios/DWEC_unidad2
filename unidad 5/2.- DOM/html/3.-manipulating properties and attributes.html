Ejemplo de manipulación de texto en DOM

innerHTML

Antes

Este texto va a ser sustituido, mediante innerHTML. InnerHTML cambiará el todos los hijos del nodo seleccionado e interpretará las etiquetas HTML. El código es el siguiente:

let inHTML=document.getElementById("inHTML"); inHTML.innerHTML="texto cambiado, mediante innerHTML, tras crear el DOM";

Después

Este texto va a ser sustituido, mediante innerHTML, usando el siguiente código:

let inHTML=document.getElementById("innerHTML"); inHTML.innerHTML="texto cambiado, mediante innerHTML, tras crear el DOM";

outerHTML

Antes

Este texto va a ser sustituido, mediante outerHTML, usando el siguiente código:

let outHTML=document.getElementById("outHTML"); outHTML.outerHTML="texto cambiado, mediante outerHTML, tras crear el DOM";

Después

Este texto va a ser sustituido, mediante outerHTML, usando el siguiente código:

let outHTML=document.getElementById("outHTML"); outHTML.outerHTML="texto cambiado, mediante outerHTML, tras crear el DOM";
