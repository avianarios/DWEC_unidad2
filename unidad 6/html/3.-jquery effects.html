Animations

basic syntax of the jQuery animate() method: $(selector).animate({ properties }, duration, callback). Only properties (CSS properties to be animated) is mandatory

elements must have relative, fixed, or absolute position to manipulate or animate its position

CSS properties names must be camel-cased when using with the animate() method, e.g. to animate the font size it has to be written 'fontSize' rather than 'font-size'
