JQuery filtering DOM

first, last, eq, filter, has, not, slice

filter can take the selector or a function as its argument to filters the set of matched elements based on a specific criteria.

$("p").has("div") returns only those p elements that has a div element as descendant

slice filters the set of matched elements specified by a range of indices
